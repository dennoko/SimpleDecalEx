# SimpleDecalEx使い方

## シェーダーの切り替え
シェーダー一覧から **dennokoworks/SimpleDecalEx** を選択してください。

## SimpleDecalEx パラメーター説明

SimpleDecalEx は lilToon のデカール仕様に準拠したデカールを **8 枚**追加するカスタムシェーダーです。  
マテリアルインスペクター上の「SimpleDecalEx」セクション内に、以下のパラメーターが表示されます。

---

## デカール 1〜8

各デカールは独立したパラメーターセットを持ちます。以下の説明で `n` は 1〜8 の番号を表します。

### 有効

| プロパティ名 | 型 | 説明 |
|---|---|---|
| `_DecalnEnable` | Toggle (0/1) | このデカールの描画を有効にします。オフにすると以下のパラメーターはすべて無視されます。 |

---

### テクスチャ / 色

| プロパティ名 | 型 | 説明 |
|---|---|---|
| `_DecalnTex` | Texture 2D | デカールとして貼るテクスチャ。アルファチャンネルでマスクを制御します。 |
| `_DecalnColor` | Color | テクスチャ全体に乗算するカラーティント。白(1,1,1,1)で元の色のまま。 |

---

### 配置

| プロパティ名 | 型 | 範囲 | 説明 |
|---|---|---|---|
| `_DecalnPosX` | Float | 0〜1 | UV 空間での横方向の位置。0.5 が中央。 |
| `_DecalnPosY` | Float | 0〜1 | UV 空間での縦方向の位置。0.5 が中央。 |
| `_DecalnScaleX` | Float | — | 横方向のスケール。1 が等倍。 |
| `_DecalnScaleY` | Float | — | 縦方向のスケール。1 が等倍。 |
| `_DecalnAngle` | Float | 0〜360 | デカールの回転角度（度数法）。 |
| `_DecalnUVMode` | Enum | 0〜3 | 参照する UV チャンネル。`UV0` / `UV1` / `UV2` / `UV3` から選択。 |
| `_DecalnMirror` | Enum | 0〜5 | ミラー挙動の設定。下表を参照。 |

#### ミラーモード一覧

| 値 | 表示名 | 説明 |
|---|---|---|
| 0 | なし | ミラー処理を行いません。 |
| 1 | 左のみ | モデルの左側（X < 0.5）にのみ表示します。 |
| 2 | 右のみ | モデルの右側（X ≥ 0.5）にのみ表示します。 |
| 3 | 左右対称コピー | 左右両側に同じデカールをコピーして表示します。 |
| 4 | 左右対称コピー（反転） | 左右両側に表示し、片方を水平反転したコピーにします。 |
| 5 | ミラーで反転 | VRChat のミラー越しに見たときに反転する挙動です。 |

---

### 合成

| プロパティ名 | 型 | 値 | 説明 |
|---|---|---|---|
| `_DecalnBlendMode` | Enum | 0〜3 | デカールの合成モード。下表を参照。 |
| `_DecalnCull` | Enum | 0〜2 | 表示する面を選択。下表を参照。 |

#### 合成モード一覧

| 値 | 表示名 | 説明 |
|---|---|---|
| 0 | 通常 | アルファブレンドで合成します（一般的なデカール）。 |
| 1 | 加算 | デカールの色を下地に加算します。光の演出に向きます。 |
| 2 | スクリーン | スクリーン合成。加算より明るくなりすぎにくい。 |
| 3 | 乗算 | デカールの色を下地に乗算します。影や汚れの表現に向きます。 |

#### カリングモード一覧

| 値 | 表示名 | 説明 |
|---|---|---|
| 0 | 両面 | 表面・裏面の両方にデカールを表示します。 |
| 1 | 表面のみ | ポリゴンの表面にのみ表示します。 |
| 2 | 裏面のみ | ポリゴンの裏面にのみ表示します。 |

---

## MatCap（全デカール領域）

有効な全デカールの表示領域の**和集合**に限定して、1 枚の MatCap を合成します。  
デカールが 0 枚の場合は何も描画されません。

| プロパティ名 | 型 | 説明 |
|---|---|---|
| `_SDEXMatCapEnable` | Toggle (0/1) | MatCap 機能を有効にします。 |
| `_SDEXMatCapTex` | Texture 2D | MatCap テクスチャ（球形マッピング用の正方形画像を推奨）。 |
| `_SDEXMatCapColor` | Color | MatCap テクスチャに乗算するカラーティント。 |
| `_SDEXMatCapBlendMode` | Enum | MatCap の合成モード。デカールと同じ 0〜3 の選択肢（通常 / 加算 / スクリーン / 乗算）。 |
| `_SDEXMatCapOpacity` | Float (0〜1) | MatCap 全体の不透明度。0 で非表示、1 で最大強度。 |
| `_SDEXMatCapMainStrength` | Float (0〜1) | MatCap の色をサーフェスのメインカラーで染める強度。0 = 染めなし、1 = 完全乗算。 |
| `_SDEXMatCapEnableLighting` | Float (0〜1) | ライティングへの追従度。0 = 陰影なしのオーバーレイ、1 = サーフェスの陰影に追従。 |
| `_SDEXMatCapShadowMask` | Float (0〜1) | 影部分での MatCap の減衰量。1 に近いほど影の中では MatCap が非表示になります。 |

---

## デカールのステッカー化（半透明用）

半透明（Transparent）マテリアルでデカール部分のアルファ値を強制的に不透明化します。  
ガラス面に貼ったシール（ステッカー）のような表現が可能です。  
**不透明 / カットアウトのレンダリングモードでは効果がありません。**

| プロパティ名 | 型 | 説明 |
|---|---|---|
| `_SDEXAlphaOverrideEnable` | Toggle (0/1) | ステッカー化機能を有効にします。 |
| `_SDEXAlphaOverrideStrength` | Float (0〜1) | デカール領域のアルファ上書き強度。1 でデカール部分を完全に不透明化します。 |
